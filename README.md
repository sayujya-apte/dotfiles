# Apte's dotfiles

### Software used
- bspwm : window manager
- sxhkd : hotkey daemon
- zsh : shell
- alacritty : terminal emulator
- picom : compositor
- polybar : status bar
- rofi : launcher
- neovim : text editor
- ranger : file manager
- starship : zsh prompt
- bat : modern cat
- lsd : modern ls
